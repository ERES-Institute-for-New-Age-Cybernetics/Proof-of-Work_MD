## Purpose

This repository packages auxiliary research, reference specifications, and technical support documents for the **PlayNAC KERNEL** and related ERES frameworks. It functions as the R&D companion to the primary [`PlayNAC-KERNEL`](../PlayNAC-KERNEL) and [`ZIP`](../ZIP) (codebase) repositories.

## What's Inside

- **Specifications** — formal `ERES-*-SPEC-*` documents (BODY, BRAINS, SPT, RCI, and related).
- **White papers** — Patriot Dividend, Transparency Deficit, ESVRD, 401(k) Soft Landing, SCALULAR Engine, BEC Architecture, GSSG/Sugar-Carbon Reservoir, Talonics Symbol Matrix.
- **Reference materials** — backronym registries, locked-terms registry (ERES TERMS #47), Three Governing Principles appendices.
- **Implementation notes** — integration guidance for subsystems (SECUIR, VERTECA, FAVORS, CBGMODD).
- **Validation records** — MIEVM scoring artifacts (Claude, Grok, DeepSeek, ChatGPT).
