## SECTION 3 — AMPLITUDE RUBRIC (Formal Scoring Criteria)

Amplitude is a two-criterion score. Criterion 1: the marker must be present. Criterion 2: the meaning must meet the tier threshold. Presence is necessary but not sufficient for a high score.

**Reproducibility requirement:** Two independent scorers applying this rubric to the same word must agree within ±1. A difference of more than 1 constitutes a falsification of inter-rater reliability (see Section 5, Test 5).

| Score | Criteria (all must hold) |
| :---- | :---- |
| **10** | Marker present. Primary referent is a proper noun for the divine, a cosmological constant, or an ERES foundational term. Meaning is non-substitutable — no synonym conveys the same function. |
| **9** | Marker present. Referent is a first-order epistemic or ethical concept. The concept is load-bearing in at least two unrelated domains. |
| **8** | Marker present. Referent is a core governance, architectural, or character term. Concept is operational — it produces action, not merely description. |
| **7** | Marker present. Referent is a meaningful supporting concept — useful but not irreplaceable in the framework. |
| **6** | Marker present. Referent is a natural or systemic phenomenon with indirect governance relevance. |
| **≤5** | Marker present but meaning is incidental, trivial, or common. Marker presence alone does not elevate score. |

**Calibration examples:**
