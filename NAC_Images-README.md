## About This Repository

**NAC_Images** is the visual asset library for the ERES Institute — a collection of AI-generated concept art, logos, dashboard mockups, formula visualizations, and framework diagrams produced through collaborative sessions with multiple AI image generation platforms. These images serve as visual companions to the text-based corpus housed across the ERES repository ecosystem.

Unlike the rest of the ERES corpus (licensed under CCAL v2.1), this repository is licensed **CC0-1.0** (public domain) for maximum reuse — anyone may use, modify, or redistribute these images without attribution for any purpose.
