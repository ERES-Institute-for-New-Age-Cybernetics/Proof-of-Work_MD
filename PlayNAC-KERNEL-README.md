## About This Repository

**PlayNAC-KERNEL** is the comprehensive PDF research archive of the ERES Institute for New Age Cybernetics — the most extensively committed repository in the ERES GitHub organization (365+ commits). It contains the full PDF corpus spanning 2012–2026: thesis documents, white papers, partnership proposals, philosophical analyses, technical specifications, conference submissions, and foundational research.

This is the **PDF companion** to [Proof-of-Work_MD](https://github.com/ERES-Institute-for-New-Age-Cybernetics/Proof-of-Work_MD) (the Markdown archive). Together, these two repositories constitute the complete proof-of-work record for the ERES framework.

> **Note:** Despite the repository name referencing "Codebase," the contents are primarily PDF research documents. The PlayNAC KERNEL codebase artifacts exist as archived packages within [Gracechain-Meritcoin](https://github.com/ERES-Institute-for-New-Age-Cybernetics/Gracechain-Meritcoin) (`ERESPlayNACKERNELCodebaseV1.zip`).
