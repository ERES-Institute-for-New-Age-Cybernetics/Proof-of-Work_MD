## Purpose

A **visual asset library** supporting the ERES corpus: logos, dashboard mockups, matrices, equation visualizations, propulsion and engineering concepts, and AI-generated concept art.

The images exist to help the framework travel — to be embedded in papers, presentations, blog posts, and teaching materials without friction.

## License Exception — Why CC0

Unlike the rest of ERES (which is CCAL v2.1), **NAC_Images is released under CC0-1.0** — full public domain dedication.

> Visual assets are released into the public domain to maximize their utility as communication tools. The images exist to spread the framework, not to protect it.

> *"Build for generations to come" — and give them the pictures to build with.*

## Embedding These Images

Raw GitHub URL pattern:

```
https://raw.githubusercontent.com/ERES-Institute-for-New-Age-Cybernetics/NAC_Images/main/<FILENAME>
```

Markdown:

```markdown
![ARI Dashboard](https://raw.githubusercontent.com/ERES-Institute-for-New-Age-Cybernetics/NAC_Images/main/ARI_Dashboard_v1.png)
```

HTML:

```html
<img src="https://raw.githubusercontent.com/ERES-Institute-for-New-Age-Cybernetics/NAC_Images/main/ARI_Dashboard_v1.png" alt="ARI Dashboard">
```

## File Naming Convention

For contributions:

```
[Framework]_[Description]_[Version].[ext]
```

Examples:

- `ARI_Dashboard_v2.png`
- `Talonics_SymbolMatrix_v1.svg`
- `SCALULAR_FourPillars_v3.png`
