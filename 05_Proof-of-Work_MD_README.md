## What This Repository Is

A flat markdown archive preserving the **genesis corpus** of ERES Institute — documents, notes, and working records spanning the CyberRAVE (pre-1997), SaleBuilders (1997–2012), and ERES Institute (2012–present) eras. The name *Proof-of-Work_MD* predates the adoption of **Proof-of-Resonance** as the canonical consensus mechanism; the repository's role is **evidentiary**, not consensus-mechanical.

> The archive is a **timestamped prior-art record**. Its markdown form is intentionally durable, diff-able, and human-readable.
